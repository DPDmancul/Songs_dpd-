# Songs_dpd-
LaTeX package to write chorded music


### Installation

    # mkdir -p /usr/share/texlive/texmf-dist/tex/latex
    # cd /usr/share/texlive/texmf-dist/tex/latex
    # git clone git@github.com:DPDmancul/Songs_dpd-.git
    # texhash
